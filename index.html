Abstract

HTML stands for HyperText Markup Language.
The "hypertext" feature is the links embedded in the text to access additional information about that text.
HTML is used to structure any text semantically speaking.
Bare HTML files, or non styled HTML, is presented by browsers with "default" styling. This default styling is called "User-Agent style".

Basic HTML structure

The HTML is based on "tags". Each tag starts with "<" (open angled bracket) and ends with ">" (closing angled ). Most of HTML tags have "closing tags" . The only difference is that the "tag name" is preceeded by "/" (a slash).
The angled brackets ("<") are not allowed in the regular text. They are limited to the tags only, in HTML. So how to display angled brackets in a web page ?
We use "HTML" symbols. An HTML symbol start with "&", ends with a ";" and it has an acronym. For example, the opening angled bracket has the acronym "lt" (Less Than). The HTML symbol for it is : &lt;

Basic tags

Here follow the description of the different basic tags.

<html>

This is the root tag for any HTML file. Any HTML content should be surrounded by that tag.

<head>

The head tag contains the "technical" information about the HTML file. It is meant to be understood by the browser or any automatic system. And it is invisible the reader.

<body>

This is the main content shown to the reader. Everything that should be displayed to the reader should be described there.

<title>

It is put inside the <head>. And it is the title of the web page displayed at the tab level of the browser.

<div>

It is just a division of the text that have no "semantic" focus or function.

Headings - <h1>, <h2>, ...<h6>

They are the section headings. The numbers are not the heading order but they mean the heading level.
In a web page, there is only one <h1>. It is the main title of the document. And in a document, we can not have several main titles. Otherwise, it is incorrect semantically.

<section>

The section is a block of paragraphs and sub-sections with a dedicated topic.

<p>

It separates the text into paragraphs. It is also a group of sentences dealing and describing the same context.

<br>

"br" stands for "line BReak". It adds a line break to the text without jumping paragraphs.

Lists

Lists, or outline, can be: ordered (numbered) or unordered (bulleted)

Ordered lists

<ol> is used for Ordered Lists.

Unordered lists

<ul> is used for Unordered Lists.

List items

They are surrounded by the tag <li>. It stand for List Item.

  3. Item 3

Emphasizing tags

Sometimes, we need to emphasize our text. And there are two ways to do it: semantically or not semantically.

  1. Semantically

    There are 2 tags for that:

    1. <strong> meaning that the text represents a strong idea,
    2. and <em> to emphasize the text and highlight it.

  2. Not semantically

    There are 3 tags for that:

    1. <b> to bold the text, <i>
    2. to italicize the text and <u>
    3. to underline the text.

Links and anchors

The HyperText feature of HTML is based on links. That's what brings additional dimensions to the text.

The main tag is <a> for "Anchor". It can be used to make internal links inside the same page. And also, it is used to make external links to other web pages. And to reference the link, we use a dedicated attribute called "href". Example:

    The referenced link or page is located outside the current web page.

  2. Internal links

    The referenced link or page is located inside the current web page. The reference is the ID of any HTML element in the page.
