HTML/CSS & Javascript Cheat Sheet

This a course cheat sheet for HTML/CSS and Javascript.

HTML
  • HTML is based on tags. We put the tag name inside < and >. To close the tag, we preceed the name with "/" and put it inside < and >.
  • Consecutive white spaces in HTML file are reduced into one unique space when it is displayed by the browser.
  • For non-breakable white spaces, we use the HTML symbol: &nbsp;.
  • <html>: The root tag
  • <head>: Dealing with the browser.
  • <body>: Visible content to the user
  • <main>: Main section of the document
  • <header>: All required header information for the web page, displayed to the user.
  • <article>: Encapsulates an article (product, magazine post, ...)
  • <section>: section tag
  • <p>: paragraph
  • <h1>: Main heading of the web page. It is not valid to put more than one H1 tag, but it is possible.
  • <h2>, <h3>, ..., <h6>: Subheadings' levels
  • <ol>: Ordered list (numbered)
  • <ul>: Unordered list (bulleted)
  • <li>: List items
  • <br>: BReak line, to split a paragraph.
  • <img>: Insert an image, with "src" attribute for the location of the image.
  • <a>: Anchor to create a link. "href" is the attribute to reference the link (Hyper-REFerence).
  • <table>: To create a table.
  • <thead>: Table header
  • <tbody>: Table body
  • <tfoot>: Table footer
  • <caption>: Table caption (legend)
  • <tr>: Table row
  • <th>: Heading cell (using for column headings or line headings)
  • <td>: Table cell or more precisely Table Data.
  • <strong>: To display a "strong" idea (in bold)
  • <em>: To emphasize a text (italic)
  • <u>: Underline text
  • <b>: Bold the text
  • <i>: Italicize the text

CSS
  • CSS stands for Cascading StyleSheet
  • Syntax: 
    selector {
    attribute: value;
    ...
}
  • The different types of selectors:
    • Tags: example body
    • Classes: They are prefixed by a dot. Example, .my-class.
    • Identifiers: They are prefixed by hashtag example #my-id.
    • Pseudo-elements
    • Pseudo-classes
  • Selectors can be combined:
    • By a "space": example nav li. It means any the <li> elements that are inside any <nav> of the HTML.
    • By ">": Refencing direct children.
    • By "~": Referencing siblings
    • By "+": Referencing the exact next sibling
  • Some attributes:
    • background-color
    • color: to color the text
    • font-family
    • display
    • font-weight
    • text-align
    • padding
    • ...
  • Be careful: padding attribute is not applicable to inline elements. It may be necessary to change the display of the inline element into an "inline-block" or "block"
  • There are many other exceptions and "weird" behaviors in CSS. The only way to overcome them is to search the Internet.
