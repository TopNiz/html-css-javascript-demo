Images in HTML

Abstract

Images are inline elements and made to display photos. For that we use the tag <img>.

Like the anchors, it requires the source of the image which is the address of the image, on the web or the local path to the image. The source attribute is "src".

Sizing the image

We use the following attributes to resize the image:

  1. "width" in pixels or in percentage (the number should followed by %)
  2. "height" in pixels or in percentage (the number should followed by %)

If no attribute is mentioned, then the image is displayed with its original size.

If one attribute is mentioned, then the browser will keep the aspect ratio.

If both attributes are mentioned, then the aspect ratio is not respected and the image may be squeezed.
