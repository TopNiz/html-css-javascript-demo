Tables in HTML

Abstract

Tables are very handy to display data and structured information. In some cases, it can be used to organize the layout of the page.

Complete usage

The main tag is <table>. And it is composed of many other elements:

  • Caption: <caption>
  • Table head: <thead>
  • Table body: <tbody>
  • Table footer: <tfoot>
  • Each of them can contain rows: <tr>
  • Each row is divided into cells: <td> (for Table Data)
  • Some cells are used as row or column headers: <th>
